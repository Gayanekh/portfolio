# Senior Portfolio Starter

Next.js App Router + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion starter for a professional portfolio.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui primitives
- Framer Motion

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize

- Edit content and sections in `components/portfolio-page.tsx`
- Add more UI components under `components/ui`
- Update metadata in `app/layout.tsx`
